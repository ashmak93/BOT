## **Быстрый запуск**



* ###### Устанавливаем Python 3.11-3.12

###### (https://www.python.org/downloads/windows/)

###### 

* ###### Устанавливаем Tesseract-OCR 

###### (https://github.com/UB-Mannheim/tesseract/wiki)

#### 

##### **pip install tesseract**



проверка:



python --version



pip --version

#### 

* ###### Заходим в папку с кодом, открываем .gitignore и удаляем tokens.txt оттуда. Затем открываем tokens.txt и вставляем bot token и gigachat token. После устанавливаем зависимости и запускаем бота:

##### 

##### **pip install -r requirements.txt**

##### 

##### **python main.py**



* ###### Бот работает!!!!!
